<h2>How to use:</h2>

<h3>Initialize</h3>
<pre> let editor=M9Editor(frame: NSMakeRect(0,0,300,300));
self.view.addSubview(editor);</pre>

<h3>Set Content</h3>
<pre>editor.setContent("<b>Hello Max9xs Here</b>")</pre>

<h3>Get Content</h3>

<pre>print(editor.getContent()); </pre>



<h2>Features</h2>
- Swift 2.0
- Bold
- Italic
- Underline
- Stroke
- Fonts
- Font-Size
