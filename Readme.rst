
HTML Editor for Cocoa


How to use:

let editor=M9Editor(frame: NSMakeRect(0,0,300,300));
self.view.addSubview(editor);
editor.setContent("<h1>Hello Max9xs Here.</h1>")
print(editor.getContent());



Features
- Bold
- Italic
- Underline
- Stroke
- Fonts
- Font-Size
